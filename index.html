¿Qué es una página web?

Una aplicación web es una aplicación que se puede abrir desde los navegadores con IP o URL.

EJEMPLOS:

Con IP: http://{direccion-ip}

Con URL: https://{nombre-de-dominio}{extension}

Con URL (sin SSL): http://{nombre-de-dominio}{extension}

Estas aplicaciones web requieren de un servidor que las esté haciendo accesibles con Apache o Nginx 24/7.

¿Qué es SSL?

Seguridad de la capa de transporte y su antecesor Secure Sockets Layer son protocolos criptográficos que proporcionan comunicaciones seguras por una red, comúnmente Internet. El protocolo TLS es el sucesor de SSL, introducido en 1999 como una versión mejorada de SSL 3.0, al principio llamado SSL 3.1.

SSL le da a tu web el HTTPS.

¿En qué lenguaje se hacen?

Las páginas web mayormente se hacen en HTML.

Ejemplo de web en HTML:

<!DOCTYPE html>
<html>
<h1>Hello, World!</h1>
</html>

¿Qué es CSS?

CSS, o Cascading Style Sheets, es un lenguaje de estilos utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, el formato y la apariencia de los elementos en una página web, como colores, fuentes, márgenes, y mucho más.

Con CSS, puedes aplicar estilos a un solo elemento, a grupos de elementos o a toda la página web, facilitando así la creación de diseños responsivos y atractivos.

Ejemplo de CSS:

body {
background-color: lightblue;
}
h1 {
    color: darkblue;
}

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite añadir interactividad y dinamismo a las páginas web. Es un lenguaje del lado del cliente, lo que significa que se ejecuta en el navegador del usuario, permitiendo manipular el contenido HTML y CSS de manera dinámica.

Con JavaScript, puedes crear funciones como animaciones, validaciones de formularios, interacciones en tiempo real y mucho más.

Ejemplo de JavaScript:

function JS() {
  alert('Hello world');
}

¿Qué es TypeScript?

TypeScript es un superconjunto de JavaScript que añade tipos estáticos y otras características modernas al lenguaje. Desarrollado por Microsoft, permite a los desarrolladores escribir código más robusto y fácil de mantener.

TypeScript se compila a JavaScript, lo que significa que puedes usarlo en cualquier entorno donde se ejecute JavaScript, como navegadores y servidores.

Ejemplo de TypeScript

function TS(): void {
  alert('Hello world');
}
TS();
